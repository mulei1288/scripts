## 功能描述

这是一个简单的 Hello World 脚本，用于向用户打招呼。支持自定义名称和详细输出模式。

## 使用方法

### 基本语法

```bash
hello.sh [选项] [名称]
```

### 使用示例

```bash
# 示例 1：基本用法（默认名称）
hello.sh

# 示例 2：指定名称
hello.sh Alice

# 示例 3：详细模式
hello.sh -v Bob

# 示例 4：调试模式
DEBUG=1 hello.sh Charlie
```

## 参数说明

### 必需参数

无

### 可选参数

- `[名称]`：要打招呼的名称（默认：World）
- `-v, --verbose`：启用详细输出
- `-h, --help`：显示帮助信息

### 环境变量

- `DEBUG`：设置为 1 启用调试日志
- `VERBOSE`：设置为 1 启用详细输出

## 依赖工具

- bash 4.0+
