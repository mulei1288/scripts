> 一个简单的 Hello World 脚本示例，展示如何使用项目模板创建脚本。

## 功能描述

这是一个简单的 Hello World 脚本，用于演示：
- 基本的脚本结构
- 参数解析
- 日志输出
- 错误处理

## 使用方法

```bash
# 基本用法
./hello.sh

# 指定名称
./hello.sh World

# 详细模式
./hello.sh -v Alice

# 调试模式
DEBUG=1 ./hello.sh Bob
```
