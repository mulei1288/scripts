# Kubernetes etcd 集群运维工具集

用于 Kubernetes etcd 集群的健康检查和故障修复的自动化脚本工具。

## 工具概览

| 工具 | 功能 | 使用场景 |
|------|------|------|
| **etcd-check** | 检测 etcd 集群异常节点 | 故障排查 |
| **etcd-repair** | 修复 etcd 集群异常节点 | 节点故障恢复 |


## 工作流程

```
┌─────────────────┐
│  etcd-check.sh  │  检测集群健康状态
│  (任意节点执行)  │  → 输出异常节点 IP
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ etcd-repair.sh  │  修复异常节点
│ (异常节点执行)   │  → 重新加入集群
└─────────────────┘
```

## 快速开始

### 1. 检测异常节点

```bash
./etcd-check.sh
```

**输出示例**：
```
# 在任意控制节点执行
[不可达/不健康/数据不一致] <异常节点IP>
```

### 2. 修复异常节点

```bash
# 在异常节点上执行
./etcd-repair.sh <异常节点IP>
```
